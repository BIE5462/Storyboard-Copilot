Storyboard-Copilot 是一个基于节点画布的 AI 分镜工作台，支持图片上传、AI 生成/编辑、裁剪/标注/分镜等工具流程。前端技术栈为 React 18 + TypeScript + Zustand + @xyflow/react + TailwindCSS，桌面容器为 Tauri 2，后端为 Rust 命令接口，数据存储为 SQLite（rusqlite, WAL），国际化使用 react-i18next/i18next。核心目录：src/features/canvas（画布主流程、节点、工具、模型、UI）、src/stores（全局状态与持久化）、src/commands（前端到 Tauri 命令桥接）、src-tauri/src（Rust 命令与 AI provider 实现）、docs/development-guides（开发文档）。项目遵循 UI -> Store -> Application Service -> Command/API -> Persistence 的分层数据流。
